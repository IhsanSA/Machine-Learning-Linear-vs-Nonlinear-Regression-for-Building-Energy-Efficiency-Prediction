# Machine-Learning-Linear-vs-Nonlinear-Regression-for-Building-Energy-Efficiency-Prediction
This machine learning project analyzes the Energy Efficiency dataset with 768 simulated samples and two targets, Heating Load and Cooling Load. Linear models show similar performance, indicating limits of linear assumptions. Non linear models, especially SVR and KNN, achieve better results, showing stronger suitability for this dataset.
Data Set : https://archive.ics.uci.edu/dataset/242/energy+efficiency
